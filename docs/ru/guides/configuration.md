# Конфигурация

Конфигурация — флаги, с fallback'ом на ENV. Файла конфига нет: секреты — в окружении или менеджере секретов.

## Приоритеты

1. CLI-флаг (например `--key`)
2. Переменная окружения
3. Встроенный дефолт (где применимо)

## Переменные окружения

| ENV                      | Маппится на         |
|--------------------------|---------------------|
| `INDEXNOW_KEY`           | `--key`             |
| `INDEXNOW_HOST`          | `--host`            |
| `INDEXNOW_KEY_LOCATION`  | `--key-location`    |
| `INDEXNOW_ENDPOINT`      | `--endpoint`        |

`INDEXNOW_ENDPOINT` срабатывает только если `--endpoint` оставлен в дефолте (`api`); явный флаг побеждает ENV.

## Вывод host

Если `--host` / `INDEXNOW_HOST` пуст, host парсится из **первого** URL в батче. По спеке все URL одного вызова должны принадлежать одному сайту.
